# Buildo Resume

> AI-оптимизация резюме за 60 секунд. ATS-friendly

Один из 6 продуктов экосистемы **Buildo** — ИИ-агентов для бизнеса.

## Что это

Резюме под вакансию — это ai-оптимизация резюме за 60 секунд. ats-friendly.

## Стек

- **Next.js 14** + React 18
- **TypeScript 5.7**
- **Tailwind CSS 3.4**
- **Brand color:** `#0EA5E9`

## Локальная разработка

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Деплой

Платформа: **Layero** (auto-deploy из main).
